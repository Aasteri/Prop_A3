## Reference Code Pattern

`AAA/GZP/530/JULY/2026/01`

| Segment | Meaning |
|---|---|
| AAA | Triple A prefix |
| GZP | Site code (GZP=Guzape, JKW=Jikwoyi, MPP=Mpape, LGB=Lugbe) |
| 530 | Project number |
| JULY/2026 | Month/year |
| 01 | Sequence (auto-increment per project per day) |
